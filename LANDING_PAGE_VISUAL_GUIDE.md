## 📐 Component Breakdown

### **1. LandingPageHeader**
```
Height: 64px (fixed)
Content: [Logo] [Nav Links] [Sign In] [Get Started]
Mobile: Hamburger menu with drawer
Animation: Fade-in on load
```

### **2. HeroSection**
```
Height: 100vh (full screen)
Layout: 2-column (1 column on mobile)
- Left: Text content + CTAs
- Right: Dashboard mockup (desktop only)
Animation: Staggered container, floating elements
```

### **3. TrustSection**
```
Height: Auto
Content: 6 brand logos + 3 stats
Grid: 2 cols → 3 cols → 6 cols (responsive)
Animation: Hover color effect on logos
```

### **4. FeaturesSection**
```
Height: Auto
Layout: 3x2 grid (6 feature cards)
Card Elements: Icon + Title + Description
Animation: Staggered fade-in, hover lift effect
```

### **5. WorkflowSection**
```
Height: Auto
Layout: Horizontal (desktop) / Vertical (mobile)
Steps: 4 (Plan → Collaborate → Review → Analyze)
Animation: Arrow connectors, hover effects
```

### **6. DashboardPreviewSection**
```
Height: Auto
Content: Large dashboard mockup
Elements: Chart, metrics, activity feed, floating cards
Animation: Chart bars animated, floating elements
```

### **7. AIFeaturesSection**
```
Height: Auto
Layout: 2-column (1 column on mobile)
- Left: Text + 4 features
- Right: Contract UI preview
Animation: Split fade-in, rotating AI badge
```

### **8. CTASection**
```
Height: Auto
Content: Headline + dual buttons + trust text
Animation: Animated gradient background, glow effects
```

### **9. Footer**
```
Height: Auto
Layout: Grid 1 → 2 → 5 columns (responsive)
Content: Brand, links, newsletter, social, copyright
Animation: Staggered items, hover effects
```

## 📱 Responsive Breakpoints

```
Mobile (< 768px)
├── 1 column layout
├── Hero: Single column
├── Features: 1-2 columns
├── Workflow: Vertical stack
├── Footer: 2 columns
└── Hamburger menu active

Tablet (768px - 1024px)
├── Multi-column layouts
├── Hero: Still 1 column
├── Features: 2-3 columns
├── Workflow: Still vertical
├── Footer: 4 columns
└── Mostly responsive

Desktop (> 1024px)
├── Full 2+ column layouts
├── Hero: True 2-column with dashboard
├── Features: 3 columns
├── Workflow: 4 columns horizontal
├── Footer: 5 columns
└── All animations active
```

## ⚡ Animation Triggers

```
Scroll Events:
├── Features fade in when scrolled into view
├── Dashboard elements stagger in
├── Stats animate with scroll
└── All use viewport={{ once: true }}

Hover Events:
├── Feature cards lift (scale + Y shift)
├── Brand logos change color
├── Buttons scale and glow
└── Links underline on hover

Continuous Animations:
├── Hero background gradients
├── Floating dashboard mockup
├── Floating accent cards
├── Pulsing badge indicators
└── Chart bar animations

Load Animations:
├── Header items stagger in
├── Hero content cascades
├── Buttons fade in
└── All use initial → animate
```

## ✨ Key Visual Patterns

### Gradient Text
```tsx
bg-gradient-to-r from-purple-400 to-blue-400
bg-clip-text text-transparent
```
Used for: Headlines, accent text

### Glass Effect
```tsx
bg-slate-800/50 backdrop-blur-xl border border-white/10
```
Used for: Cards, sections, backgrounds

### Hover Lift
```tsx
whileHover={{ y: -8, scale: 1.02 }}
```
Used for: Feature cards, workflow steps

### Gradient Border Glow
```tsx
hover:border-purple-500/30 transition-colors
```
Used for: Interactive elements

### Floating Animation
```tsx
animate={{ y: [0, -20, 0] }}
transition={{ duration: 4, repeat: Infinity }}
```
Used for: Dashboard, floating cards

## 🎬 Animation Flow Timeline

```
Page Load (0ms)
├─ Header: Fade in (0-500ms)
├─ Hero: Stagger content (100-800ms)
├─ Background: Start gradient animation (0ms, infinite)
└─ Dashboard mockup: Float up (300-1100ms)

User Scrolls → Section enters Viewport
├─ Container fades in (0-600ms)
├─ Items stagger in (0-1000ms total)
└─ Animations complete (once: true)

User Hovers over Interactive Element
├─ Element transforms (0-300ms)
├─ Color changes (smooth transition)
└─ Return on mouse out (0-300ms)

Continuous (All sections)
├─ Background glows animate (6-8s cycle)
├─ Floating elements move (4s cycle)
├─ Pulsing indicators animate (2s cycle)
└─ All ease-in-out for smoothness
```
